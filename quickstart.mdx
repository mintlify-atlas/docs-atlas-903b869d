Get Dashi running on your local machine in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- **Node.js 20+** or **Bun** installed
- **Rust** and **Cargo** (for building the Tauri backend)
- **Groq API key** for voice transcription
- **Letta server** running locally (optional but recommended for AI features)

## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/ehcaw/dashi.git
    cd dashi
    ```
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>
    ```bash npm
    npm install
    ```

    ```bash bun
    bun install
    ```

    ```bash pnpm
    pnpm install
    ```

    ```bash yarn
    yarn install
    ```
    </CodeGroup>
  </Step>

  <Step title="Set up environment variables">
    Create a `.env` file in the `src-tauri` directory:

    ```bash
    GROQ_API_KEY=your_groq_api_key_here
    ```

    Get your Groq API key from [groq.com](https://groq.com).
  </Step>

  <Step title="Start the Letta server (optional)">
    If you want to use the AI chat features, start a Letta server:

    ```bash
    # Install Letta
    pip install letta

    # Start the server
    letta server --port 8283
    ```

    The default configuration in `src/lib/letta.ts` expects the server at `http://localhost:8283`.
  </Step>

  <Step title="Run the development server">
    <CodeGroup>
    ```bash npm
    npm run dev
    ```

    ```bash bun
    bun dev
    ```

    ```bash pnpm
    pnpm dev
    ```

    ```bash yarn
    yarn dev
    ```
    </CodeGroup>

    This starts the Next.js development server at [http://localhost:3000](http://localhost:3000).
  </Step>

  <Step title="Launch the desktop app">
    To run Dashi as a desktop application with Tauri:

    <CodeGroup>
    ```bash npm
    npm run tauri dev
    ```

    ```bash bun
    bun tauri dev
    ```

    ```bash pnpm
    pnpm tauri dev
    ```

    ```bash yarn
    yarn tauri dev
    ```
    </CodeGroup>

    The Tauri window will open with the Dashi interface.
  </Step>
</Steps>

## First Interaction

Once Dashi is running, you can interact with it:

<Steps>
  <Step title="Click to activate">
    Click on the Dashi avatar to activate it. The opacity will increase from 0.1 to 0.3.
  </Step>

  <Step title="Try voice mode">
    Press and hold **Cmd+Option+V** (macOS) or **Ctrl+Alt+V** (Windows/Linux) to start voice recording. Release to stop recording and get a transcription.

    <Note>
      The first time you use voice mode, your browser will ask for microphone permission.
    </Note>
  </Step>

  <Step title="Try chat mode">
    Press **Cmd+Option+T** (macOS) or **Ctrl+Alt+T** (Windows/Linux) to open the chat interface. Type a message and press **Send** or hit **Enter**.
  </Step>

  <Step title="Reset">
    Press **ESC** to reset Dashi to its default state at any time.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Voice recording not working">
    - Make sure you've granted microphone permissions in your browser
    - Check that you have a valid Groq API key in `src-tauri/.env`
    - Verify the Rust backend is running (it starts automatically with `tauri dev`)
  </Accordion>

  <Accordion title="Chat not responding">
    - Ensure the Letta server is running at `http://localhost:8283`
    - Check the agent ID in `src/lib/letta.ts` matches your Letta agent
    - Look for errors in the browser console
  </Accordion>

  <Accordion title="Desktop app won't start">
    - Make sure Rust and Cargo are installed: `cargo --version`
    - Try rebuilding: `cargo clean` in the `src-tauri` directory
    - Check that all Rust dependencies can be compiled
  </Accordion>

  <Accordion title="Next.js build errors">
    - Delete `node_modules` and reinstall dependencies
    - Clear Next.js cache: `rm -rf .next`
    - Make sure you're using Node.js 20 or higher
  </Accordion>
</AccordionGroup>
