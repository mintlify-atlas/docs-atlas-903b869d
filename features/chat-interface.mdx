## Overview

Dashi's chat interface provides a modern, interactive way to communicate with your AI assistant. Activated with **Cmd+Opt+T**, it features a sleek design with gradient colors, streaming responses, and persistent message history during your session.

## Activating Chat Mode

There are two ways to enter chat mode:

1. **Keyboard shortcut**: Press **Cmd+Opt+T** (or **Ctrl+Alt+T** on Windows/Linux)
2. **Test mode**: Press **T** key (development/testing)

From `src/components/ui/dashimain.tsx:179-189`:

```typescript
const isTKey = event.key === "t" || event.key === "T" || event.keyCode === 84;
if ((event.metaKey || event.ctrlKey) && event.altKey && isTKey) {
  if (isActivated && !isActivatedState) {
    setIsActivatedState(true);
    setIsExpanded(false); // Reset voice expansion
  }
}
```

<Note>
Chat mode automatically disables voice expansion mode when activated, ensuring a clean interface transition.
</Note>

## Sending Messages

### Text Input

1. Type your message in the input field
2. Press **Enter** or click the **Send** button
3. Your message appears immediately in the chat
4. The input field clears automatically

## Streaming Responses

Dashi responses stream in real-time, creating a natural conversation flow:

1. An empty message bubble appears immediately
2. Text streams in character by character
3. The message updates in place as chunks arrive

From `src/components/ui/dashimain.tsx:312-329`:

```typescript
const dashiResponseStream = await generate_response(chatInput);
setChatMessages((prev) => [...prev, { text: "", isUser: false }]);

for await (const chunk of dashiResponseStream) {
  if (chunk.content && chunk.content !== "undefined") {
    setChatMessages((prev) => {
      const updated = [...prev];
      if (updated.length > 0) {
        updated[updated.length - 1] = {
          ...updated[updated.length - 1],
          text: (updated[updated.length - 1].text || "") + chunk.content,
        };
      }
      return updated;
    });
  }
}
```

## Message History

During your session:
- All messages persist in the chat history
- Scroll to view previous messages
- Messages clear when you press **ESC** to reset Dashi

## Resetting Chat

Press **ESC** to reset Dashi to default mode:
- Chat interface fades out
- All message history clears
- Dashi returns to idle state
- Black box shrinks back to 100px × 100px

From `src/components/ui/dashimain.tsx:130-145`:

```typescript
const handleReset = () => {
  setIsResetting(true);
  setPositionsSetToDefault(false);
  
  setTimeout(() => {
    setIsActivated(false);
    setIsExpanded(false);
    setIsActivatedState(false);
    setHasMovedLeft(false);
    setVoiceResponse("");
    setIsGeneratingVoiceResponse(false);
    setPositionsSetToDefault(true);
    setIsResetting(false);
  }, 800); // Match fade out duration
};
```

<Warning>
Resetting Dashi will clear all message history. There is no undo function.
</Warning>

## State Management

The chat interface uses these key state variables:

- `chatInput`: Current input field value
- `chatMessages`: Array of message objects with `text` and `isUser` properties
- `isActivatedState`: Boolean indicating chat mode is active
- `isResetting`: Boolean for fade out animation

## Integration with AI

Chat messages are processed by the Letta AI agent via `src/lib/letta.ts:9-20`:

```typescript
export const generate_response = async (input: string) => {
  const stream = await letta_client.agents.messages.createStream(agent_id, {
    messages: [
      {
        role: "user",
        content: input,
      },
    ],
    streamTokens: true,
  });
  return stream;
};
```
