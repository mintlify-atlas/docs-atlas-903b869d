---
title: AnimatedVectorBox
description: Interactive animated component that displays a vector-based avatar with voice recording, chat interface, and multiple interaction modes.
---

The `AnimatedVectorBox` component is the main interactive element in Dashi that provides voice recording, chat interface, and animated vector graphics capabilities.

## Import

```tsx
import AnimatedVectorBox from "@/components/ui/dashimain";
```

## Interface

```typescript
interface AnimatedVectorBoxProps {
  width?: number;
  height?: number;
  vectorPath?: string;
  vectorPaths?: string[];
  vectorColor?: string;
  boxColor?: string;
  borderRadius?: number;
  leftMargin?: number;
}
```

## Props

<ParamField path="width" type="number" default="100">
  Width of the component in pixels.
</ParamField>

<ParamField path="height" type="number" default="100">
  Height of the component in pixels.
</ParamField>

<ParamField path="vectorPath" type="string" default='""'>
  Single SVG path string for the vector graphic. Use this for a single-path avatar.
</ParamField>

<ParamField path="vectorPaths" type="string[]" default="[]">
  Array of SVG path strings for multi-path vector graphics. Takes precedence over `vectorPath` when provided.
</ParamField>

<ParamField path="vectorColor" type="string">
  Color for the vector graphic. Component uses a gradient from `#D500D8` to `#8CEBE5` by default.
</ParamField>

<ParamField path="boxColor" type="string" default='"black"'>
  Background color of the container box.
</ParamField>

<ParamField path="borderRadius" type="number" default="16">
  Border radius of the container box in pixels.
</ParamField>

<ParamField path="leftMargin" type="number" default="20">
  Left margin offset for positioning elements in pixels.
</ParamField>

## Features

### Interaction Modes

The component supports three interaction modes:

1. **Default Mode**: Initial state with low opacity (0.1)
2. **Voice Mode**: Activated with `Cmd/Ctrl + Option + V` - Expands to show voice recording interface with animated music notes
3. **Chat Mode**: Activated with `Cmd/Ctrl + Option + T` - Expands to 400px width and 600px height with a full chat interface

### Keyboard Shortcuts

- **First Click**: Activates the component (opacity increases to 0.3)
- **Cmd/Ctrl + Option + V**: Hold to activate voice recording mode with visual feedback
- **Cmd/Ctrl + Option + T**: Toggle chat interface mode
- **ESC**: Reset to default mode

### Animation Features

- Initial Lottie animation on component mount
- Smooth transitions between states (0.5s duration)
- Animated vector paths with gradient fills
- Music note animations during voice recording
- Fade in/out transitions with 0.8s duration during reset

### Chat Interface

When in chat mode, the component displays:

- Message history with user and assistant messages
- Text input field with placeholder "Type your message..."
- Send button for submitting messages
- Auto-scrolling message container (420px height)
- Streaming response support with real-time message updates

### Voice Recording

Integrates with `useVoiceRecorder` hook for:

- Real-time voice recording
- Automatic transcription on release
- Visual recording indicator with gradient border
- Voice response generation and display
- Streaming response text next to Dashi avatar

## Styling

The component uses:

- Framer Motion for animations
- Linear gradient from `#D500D8` (top) to `#8CEBE5` (bottom) for vector graphics
- User messages: `#8CEBE5` background with black text
- Assistant messages: `#D500D8` background with white text
- Semi-transparent backgrounds with blur effects
- Box shadow effects during voice recording

## Dependencies

- `framer-motion`: Animation library
- `lottie-react`: Lottie animation support
- `@/lib/tts`: Voice recording hook (`useVoiceRecorder`)
- `@/lib/letta`: Response generation (`generate_response`)
