The `TwoDifferentVectors` component (also known as MusicNotes) displays two animated SVG paths that alternate between colors and sizes, creating a visual music note effect during voice interactions.

## Import

```tsx
import TwoDifferentVectors from "@/components/ui/musicnotes";
```

## Component Details

This component is used internally by `AnimatedVectorBox` to display animated music notes when voice recording is active.

### Animation Behavior

- Two SVG music note icons alternate between cyan (#8CEBE5) and magenta (#D500D8)
- Icons scale between 100% and 120% size
- Animation cycle: 1 second interval
- Uses Framer Motion for smooth transitions

## Implementation

```typescript src/components/ui/musicnotes.tsx
export default function TwoDifferentVectors() {
  // Animation state for vector 1
  const [anim1, cycle1] = useCycle(
    { d: path1, fill: "#8CEBE5" },
    { d: scalePath(path1, 1.2), fill: "#D500D8" }
  );

  // Animation state for vector 2
  const [anim2, cycle2] = useCycle(
    { d: scalePath(path2, 1.2), fill: "#D500D8" },
    { d: path2, fill: "#8CEBE5" }
  );

  useEffect(() => {
    const interval = setInterval(() => {
      cycle1();
      cycle2();
    }, 1000);
    return () => clearInterval(interval);
  }, [cycle1, cycle2]);

  // Renders two overlapping SVG elements with animated paths
}
```

## Helper Functions

### scalePath

Scales all numeric values in an SVG path string by a given factor:

```typescript
function scalePath(path: string, scale: number): string {
  return path.replace(/([0-9]*\.?[0-9]+)/g, (match: string) => {
    const num = parseFloat(match);
    return isNaN(num) ? match : (num * scale).toFixed(4);
  });
}
```

## Animation Timing

- **Interval**: 1000ms (1 second)
- **Transition duration**: 500ms
- **Easing**: `easeInOut`
- **Fade in delay**: 300ms (when appearing)
- **Fade out duration**: 800ms (when resetting)

## Color Scheme

Matches Dashi's brand gradient:
- **Cyan**: `#8CEBE5` (primary color)
- **Magenta**: `#D500D8` (light color)

The icons continuously swap these colors while animating.

## Visual Context

The MusicNotes component appears:
- When voice recording mode is active (Cmd+Option+V held)
- Positioned to the right of the Dashi avatar
- Above the third animated path element
- Hidden when voice response is being generated
- Hidden in chat mode

This creates a visual indicator that the microphone is recording and processing audio.
