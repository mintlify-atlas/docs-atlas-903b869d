# Welcome to Dashi

Dashi is an intelligent desktop companion that brings AI conversations to your desktop. Built with Next.js 15 and Tauri, Dashi provides seamless voice and text-based interactions through an elegant, animated interface.

## What is Dashi?

Dashi is a Next.js application wrapped in a Tauri desktop shell, creating a native desktop experience for AI interactions. The interface features an animated character that responds to your interactions with smooth transitions and visual feedback.

### Built With

- **Next.js 15.3.4** - React framework with App Router
- **Tauri 2.5** - Desktop application framework
- **Letta AI** - Conversational AI backend
- **Framer Motion** - Animation library
- **React 19** - UI framework
- **TypeScript** - Type-safe development

## Keyboard Shortcuts

Dashi is designed for quick access with keyboard shortcuts:

- **Cmd+Opt+V** - Activate voice mode (hold to record, release to send)
- **Cmd+Opt+T** - Switch to chat mode
- **ESC** - Reset to default mode
